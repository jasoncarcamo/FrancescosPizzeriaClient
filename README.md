## Features
Users can browse through the restaurants menu and order online from their mobile phones. Features include ordering pick up or delivery, registering users for futures orders, and  cancelling orders.
Tracker coming soon...

## Built with
- React.js
- Express.js
- PostgreSQL
- RESTful API
- ASP.NET CORE
- JSON Web Tokens
- Bcrypt.js
- XSS.js
